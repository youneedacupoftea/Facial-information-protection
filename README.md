# Facial-information-protection
This system protects facial information in short videos and images.

## 1. Function
(1) Detects faces in images. For videos, it processes each frame and then feeds the face image into a pre-trained model to obtain the corresponding facial expression result;

(2) For face replacement in images, users can use the mouse to select any face to be replaced;

(3) Videos can be selected from local offline videos or real-time videos obtained from a computer webcam;

(4) Saves the replaced images and videos. Videos are saved frame by frame;

(5) A graphical user interface is created using the tkinter library to integrate functions and facilitate use.
